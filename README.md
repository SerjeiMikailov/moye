# "Moye" Project

### C Kernel

My objective with this project is explore the keyboard connection between the user and hardware

### Info:

- 32bits
- Bootloader: Grub
- Compiler and linker: GCC
- Stack Memory: 512 bytes (just the necessary)

### --> You can run it with QEMU
